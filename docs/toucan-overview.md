# Toucan

## Overview

Toucan is a party games app featuring multiple game modes built around prompts and social interactions. The app uses animated backgrounds and smooth transitions to create distinct visual experiences for each prompt type.

## Game Modes

### Toucan Drink
- The original game mode
- Select a deck and start playing
- Prompts displayed one at a time, tap to advance
- Drinking game with various prompt types

### It Takes Tou
- 2-player intimate conversation game
- Ask personal questions to get to know each other
- 3 progressive stages/decks that get more real and intimate
- Uses SunriseBackground visual theme

### Toucan Discuss
- Multiplayer discussion game
- Players take turns
- 3 prompts displayed at once, player selects which to read aloud
- Deep prompts designed to foster discussion and reflection

### Toucan Charades
- Phone-to-forehead guessing game
- Played in landscape mode
- 60-second timed rounds
- Controls:
  - Tilt phone DOWN = correct guess, next prompt
  - Tilt phone UP = pass, next prompt
  - Shake phone = skip (alternative control)
- Various themed decks available

## Prompt Types & Visual Themes

| Prompt Type | Animation Effect |
|-------------|------------------|
| Normal | Gentle floating particles with subtle teal gradient shift |
| Minigame | Bouncing geometric shapes with rotating animation on orange/yellow gradient |
| Virus | Pulsing red gradient with spreading virus spots that grow and contract |
| Virus Cure | Refreshing wave lines flowing across blue gradient with shimmer effect |
| Bottoms Up | Rising bubbles with wobble animation on purple gradient |
| Punishment | Dark flickering/lightning effect with random flashes and vignette |
| Spicy | Rising flame particles (yellow to orange to red) with heat shimmer |

## Visual Design

- **Transitions:** Crossfade with 500ms animation between different prompt types
- **Consistency:** Same prompt type continues animation seamlessly
- **Content Priority:** Animations are subtle to keep prompts readable
- **Mode-Specific:** Each game mode can have its own background style

## Color Palette (from prompt types)

- Teal (Normal)
- Orange/Yellow (Minigame)
- Red (Virus)
- Blue (Virus Cure)
- Purple (Bottoms Up)
- Dark with flashes (Punishment)
- Yellow/Orange/Red gradient (Spicy)
